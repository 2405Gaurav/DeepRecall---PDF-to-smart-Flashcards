# `.brain` — local context (How To AI style)

This folder is the **Brain** for *this repo*: the PDF’s idea is *your files are the AI’s memory*. Here that means Cursor / Opus reads the same notes every session.

| File | Role (maps to guide) |
|------|------------------------|
| `config.json` | **Setup** — how this workspace mirrors Part 0 (Brain root, tooling). |
| `context.md` | **Context** — what the product is, stack, decisions (Part 1). |
| `prompt` | Standing instructions = lightweight “agent” defaults (Part 2 vibe). |
| `prompts-journal.md` | Raw *WHAT* I asked, messy wording preserved on purpose. |
| `ignore` | Paths the assistant should skip so context stays clean (like not pasting secrets). |

**Not** using Claude Code CLI here — **Cursor** pointed at the project, with the same **WHAT / HOW** split the PDF describes.
